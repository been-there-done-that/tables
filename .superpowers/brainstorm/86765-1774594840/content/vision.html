What should this agent become?

Three visions — pick the one that excites you most (or mix them)

🔬

Data Analyst

The agent as a tireless analyst. You ask a question in plain English — it explores the schema, samples data, runs queries, and returns insights with inline results.

"Which users signed up last month but never placed an order?"
"Show me the top 10 products by revenue this quarter"
"Are there any orphaned records in the orders table?"
Tools
explore_schema run_query sample_table count_rows get_distinct_values
🛠

DBA Copilot

A senior DBA in your pocket. Understands schema health, index usage, FK integrity, query performance. Can propose migrations, explain EXPLAIN plans, and write DDL.

"Why is this query slow? Here's the EXPLAIN output..."
"What indexes are missing on the orders table?"
"Generate a migration to add soft deletes to users"
Tools
get_indexes explain_query check_fk_integrity get_table_stats run_ddl
✈️

Universal Copilot

All of the above, plus deep agentic loops. The agent autonomously explores, queries, cross-references tables, builds context, then delivers a complete answer — like a junior developer doing research.

"Build me a full data quality report for this database"
"Understand the schema and suggest a reporting model"
"Find all the queries from history that could be optimized"
Tools
everything above + open_in_editor query_history save_snippet multi_connection
