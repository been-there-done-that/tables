Which tools go into v1?

Pick everything you want in the first version — you can select multiple

🗂 Schema Exploration

list_tables

List all tables in a schema with row counts & types

describe_table

Full column details, types, nullable, defaults, PKs

get_indexes

All indexes on a table (name, columns, unique, type)

get_foreign_keys

FK relationships — what references what

⚡ Query Execution

run_query

Execute any readonly SELECT — agent sees the results

sample_table

Grab N random rows to understand data shape

count_rows

Fast COUNT with optional WHERE — good for sanity checks

explain_query

Run EXPLAIN / EXPLAIN ANALYZE and parse the plan

📊 Data Profiling

column_stats

NULL %, distinct count, min/max/avg for a column

find_nulls

Which columns have unexpected NULLs across a table

get_distinct_values

Top N distinct values for a column — great for enums

check_fk_integrity

Find orphaned FK rows — data quality check

🔗 Editor Integration

open_in_editor

Push a SQL query into a new editor tab

get_query_history

Fetch recent queries run in the editor — agent learns from your work
