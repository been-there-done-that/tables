AgentComposer — Bottom Bar Layout

3 options for fitting provider + bigger icons into the existing bottom bar. Icons are currently 11px; all options bump to 13–14px.

Provider replaces model — single picker

Provider + model are one combined dropdown. "Claude · Sonnet 4.6" or "Gemini · Pro". Effort stays separate. Cleanest bar — one fewer button.
