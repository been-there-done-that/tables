What's inside the OpenRouter tab?

You'll enter an API key and browse/select models. How much control do you want over the model list?

A

Key + fetch + searchable model table

Enter API key → click "Fetch models" → get the live list from OpenRouter API. Searchable table shows all models with context window size and pricing. Toggle checkboxes to pin models to the main picker.

B

Key + base URL + manual model list

API key + base URL field (for custom endpoints / self-hosted). No live fetch — you type in model IDs manually (or paste a list). Simpler, works offline, supports any OpenAI-compatible endpoint.

Has base URL field (works with any OpenAI-compatible endpoint, not just OpenRouter). Click "Fetch models" to pull the live list and browse. If fetch fails, falls back to manual entry. Google tab also fetches its model list.
